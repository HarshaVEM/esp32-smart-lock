# ESP32 Smart Lock

Built a password-based smart lock using ESP32, keypad, and LCD.

## Features
- Password authentication system
- LCD feedback for user input
- Relay-based locking mechanism

## Components
- ESP32
- 4x4 Keypad
- 16x2 LCD
- Relay Module

## Working
User enters a 4-digit password via keypad → system verifies input → 
if correct → relay activates → door unlocks → else access denied
